Define Favicon and give an illustrative example.
Ans->
A favicon that stand for a favorite icon is a tiny icon included along with a website, which is displayed in place
like the browser's address bar, page tags, and bookmarks menu. It is usually 16 * 16 pixexl in size and stored in 
GIF, PNG, or ICO file format.
They are used to improve user experience and enface brand consistency. When a familiar icon is seen in the browser
address bar.
